# Silly Selfbot
> *Best free open source discord raiding selfbot*

# Setup
1. Run `install.bat`
2. Put your token in `Settings/config.json`
3. Run `open.bat`

# Put this in your config.json file in Settings folder
```json
{
    "TOKEN" : "YOUR-TOKEN",
    "PREFIX" : ".",
    "IPLOOKUP-API-KEY" : ""
}```
